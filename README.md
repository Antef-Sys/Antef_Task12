# 📊 Compound Interest Calculator

This is a simple JavaScript project that calculates **compound interest** using the standard mathematical formula.

---

## 🚀 Project Overview

This project demonstrates how to calculate the final amount after applying compound interest over a period of time using JavaScript.

It uses the formula:

[
A = P (1 + R/N)^{NT}
]

---

## 🧮 Formula Explanation

* **A** → Final Amount
* **P** → Principal Amount (Initial Investment)
* **R** → Annual Interest Rate (in decimal form)
* **N** → Number of times interest is compounded per year
* **T** → Time (in years)

## 💻 Code Description

### HTML (`index.html`)

* Basic structure of the webpage
* Links the JavaScript file

### JavaScript (`script.js`)

* Stores values for principal, rate, time, and frequency
* Calculates compound interest using formula
* Displays result in the console

---

## 📌 Example Values Used

```js
let P = 6000;   // Principal Amount
let R = 0.08;   // Rate of Interest (8%)
let N = 4;      // Compounded quarterly
let T = 2;      // Time in years
```

## ▶️ How to Run the Project

1. Create two files:

   * `index.html`
   * `script.js`

2. Copy the HTML code into `index.html`

3. Copy the JavaScript code into `script.js`

4. Open `index.html` in your browser

5. Open **Developer Console**:

   * Press `F12` or `Ctrl + Shift + I`
   * Go to **Console tab**

6. You will see the output like:

```
The Compound interest after 2 years is: 7024.49
```
